Context Recall Changelog

Track all updates and improvements to the Context Recall extension

0.0.1 Initial Release
✨ Added
  • Task Profile Creation and Management: Create and manage profiles that capture your complete workspace state
    • Capture open files and their exact cursor positions
    • Save active editor state
    • Associate profiles with Git branches
    • Add custom notes and descriptions to profiles
  • Seamless Context Switching: Switch between task profiles with full workspace restoration
    • Automatically open saved files
    • Restore cursor positions exactly where you left off
    • Checkout associated Git branches
    • Apply Git stashes automatically
  • Smart Git Integration: Deep integration with your Git workflow
    • Automatic profile creation when creating new branches
    • Auto-switch to profiles when changing branches via command line
    • Git stash support to preserve uncommitted changes
    • Automatic cleanup when branches are deleted
  • Flexible Automation Modes: Configure how Context Recall behaves
    • Task Creation: Manual, Prompt on new branch, or Automatic on new branch
    • Profile Switching: Manual, Prompt when task exists, or Automatic when task exists
    • Task Deletion: Manual or Automatic on branch deletion
  • Profile Configuration Options: Customize default behavior for new profiles
    • Auto-update: Automatically save changes when files are modified
    • Create stash on switch: Preserve uncommitted changes
    • Auto-checkout branch: Automatically checkout associated Git branch
    • Close editors on switch: Clean up editors when switching away
  • Tree View Sidebar: Dedicated sidebar for profile management
    • View all task profiles at a glance
    • Quick access to create, switch, edit, and delete profiles
    • Visual indicators for active profile
    • Settings access directly from sidebar
  • Status Bar Integration: Quick status and access from the status bar
    • Shows current active profile
    • Click to quickly switch profiles
  • Command Palette Integration: Full command palette support
    • Context Recall: Create New Task Profile
    • Context Recall: Switch Task Profile
    • Context Recall: Edit Task Profile
    • Context Recall: Delete Task Profile
    • Context Recall: Open Settings
