Overview

PromptMD adds a full markdown toolbar and live preview directly to ChatGPT and Google Gemini. Format your prompts with bold, italic, lists, code blocks, and more — without memorising any syntax. See exactly how your prompt will render before you send it.

Key Features

🛠️

Markdown Toolbar

One-click buttons for bold, italic, headings, lists, code blocks, and more.

↵

Enter for New Line

Press Enter to add a new line instead of submitting — just like a real editor.

⇥

Tab to Indent

Use Tab to indent inside code blocks and nested lists naturally.

👁️

Live Preview

Toggle a rendered preview of your markdown before sending your prompt.

📋

Smart List Continuation

Hitting Enter inside a list automatically adds the next list item.

⌨️

Keyboard Shortcuts

Ctrl+B for bold, Ctrl+I for italic — standard shortcuts that just work.
