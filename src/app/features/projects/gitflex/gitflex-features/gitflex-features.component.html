GitFlex Features Guide

Everything GitFlex can do for your GitHub workflow

Pull Request Enhancements

⚠️

PR Age Warnings

Colour-coded age indicators on PR cards warn you when a review is overdue based on configurable thresholds.

📅

Review Day Indicators

See how many business days a PR has been open, directly on the PR list.

👥

Participant Avatars

See who has commented on or reviewed a PR at a glance, without opening it.

🪟

Split-Preview Panel

View the PR description alongside the diff in a side-by-side layout.

Issue Management

📊

Sub-Issue Progress

Recursive progress bars on parent issues show how many sub-issues are closed.

🍞

Parent Breadcrumbs

Navigate up the issue hierarchy from any sub-issue page.

🏷️

Label Management

One-click label removal and sub-issue label display without extra clicks.

📋

Bulk Actions

Act on multiple sub-issues simultaneously from the parent issue view.

Global Tools

✏️

Global Issue Creator

Press Ctrl+Shift+K anywhere on GitHub to open a full issue creator — works across repos and custom domains.

🔒

Multi-Account Support

Switch between multiple GitHub accounts without browser profile juggling.

🔗

Copy as Link

Copy the current issue or PR title as a formatted Markdown or Teams link with one click.

📊

Project Board Field Sums

Automatically sum numeric custom fields on project boards using batched GraphQL queries.

Configuration

Every GitFlex feature can be toggled independently from the extension popup. Set your own PR age thresholds, choose which topbar sections appear, configure PAT-gated features, and more. GitFlex is designed to stay out of your way when you don't need it.
