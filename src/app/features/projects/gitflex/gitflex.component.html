Overview

GitFlex is a Manifest V3 browser extension that seamlessly integrates into GitHub to give you a better-than-default experience. From smarter pull request views and enhanced comment tools to a global issue creator and multi-account support — GitFlex makes every GitHub interaction faster and more productive.

Key Features

Features Guide Changelog
🚀

Enhanced Pull Requests

PR age warnings, review day indicators, participant avatars, and a split-preview panel.

📋

Issue Organization

Sub-issue progress bars, parent breadcrumbs, label management, and bulk actions.

💬

Comment Tools

One-click copy, tab-to-indent in editors, smart quick-mention insertion.

👥

Team Collaboration

Multi-account support and a global issue creator (Ctrl+Shift+K) for any org.

⚡

Workflow Optimization

Recursive field sums on project boards, batched GraphQL queries, loading indicators.

🎨

UI Improvements

Topbar reorganization, unobtrusive affordances, and fully configurable settings.
