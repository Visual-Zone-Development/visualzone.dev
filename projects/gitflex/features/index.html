‚ö° Global Issue Creator

Create GitHub issues from anywhere with a keyboard shortcut

  • Keyboard Shortcut: Press Ctrl+Shift+K (Cmd+Shift+K on Mac) from any GitHub page
  • Repository Selection: Search and select any repository you have access to
  • Template Support: Automatically loads and applies issue templates
  • Rich Metadata: Add labels, assignees, milestones, and projects
  • Image Upload: Drag and drop or paste images directly
  • Console Logs: Attach captured console errors and warnings
  • URL Append: Automatically include the current page URL
  • Smart Memory: Remembers your last used repository
  • Security: Built-in XSS protection and input sanitization

How to Use:

  1. Set up your GitHub Personal Access Token in Authentication tab
  2. Press Ctrl+Shift+K (or Cmd+Shift+K on Mac) from any GitHub page
  3. Fill out the issue details and click "Create Issue"

‚ö° Personal Access Token Setup

Many powerful features require a GitHub Personal Access Token to access GitHub's API. Setting up a PAT unlocks enhanced analytics, API-powered insights, automated suggestions, and real-time data access.

How to Set Up Your PAT:

  1. Go to GitHub ‚Üí Settings ‚Üí Developer Settings ‚Üí Personal Access Tokens ‚Üí Tokens (classic)
  2. Click "Generate new token (classic)"
  3. Add a note (e.g., "GitFlex Extension")
  4. Set expiration (recommended: 90 days or No expiration)
  5. Select the following scopes:
    • repo - Full control of private repositories
    • read:org - Read org and team membership
    • read:user - Read user profile data
    • user:email - Access user email addresses
  6. Click "Generate token"
  7. Copy the token (you won't see it again!)
  8. Open GitFlex Settings ‚Üí Authentication tab
  9. Paste your PAT and click "Validate & Save"

Note: Your PAT is stored securely in your browser's local storage and is never transmitted anywhere except GitHub's API.
