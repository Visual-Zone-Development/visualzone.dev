Configuration Options

Automation Settings

  • Task Creation Mode: Manual, Prompt on new branch, or Automatic on new branch
  • Task Deletion Mode: Manual or Automatic on branch deletion
  • Profile Switch Mode: Manual, Prompt when task exists, or Automatic when task exists
  • Prompt on Branch Switch: Prompt to delete profile when switching away from a branch

Profile Defaults

  • Auto Update: Automatically save changes when files are modified (default: true)
  • Create Stash on Switch: Create git stash when switching away from this profile (default: true)
  • Auto Checkout Branch: Automatically checkout the associated git branch when switching to this profile (default: true)
  • Close Editors on Switch: Close all open editors when switching away from this profile (default: false)
