Overview

GamePack is an online platform that makes it easy for friends and family to stay connected through fun multiplayer party games. Built during the pandemic to help people stay social at a distance, it offers a seamless experience with no sign-up required.

Whether you're hosting a virtual game night, looking to play with friends across the globe, or just want a casual gaming session, GamePack provides an accessible and engaging platform with integrated video chat and a variety of popular party games.

Technical Implementation

Frontend

  • Angular (TypeScript)
  • Angular Material UI
  • RxJS for reactive programming
  • Canvas API for drawing games
  • WebRTC for video chat

Backend

  • Node.js & Express
  • Socket.io for real-time communication
  • MongoDB for data persistence
  • JWT authentication
  • Stripe payment integration

Infrastructure

  • Docker & Docker Compose
  • Heroku deployment
  • MongoDB Atlas
  • Automated database seeding
  • Hot-reload development setup

Platform Features

Real-Time Multiplayer

Built on Socket.io for instant, synchronized gameplay. All players see updates in real-time, making the experience smooth and responsive even with multiple participants.

No Sign-Up Required

Create a room instantly and share the code with friends. Optional account creation available for those who want to save progress and access premium features.

Room-Based Architecture

Each game session exists in its own room with a unique code. Supports public and private rooms, host controls, and easy player management.

Integrated Video Chat

WebRTC-powered video chat lets you see and hear your friends while playing. Optional feature that can be toggled on or off per room.

Premium Content

Optional store with additional game categories and features. Stripe integration for secure payments and account-based purchases.
